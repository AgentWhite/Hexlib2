# HexLib & ASL Documentation

**HexLib** is a platform-agnostic hex-grid geometry library. **ASL** builds on it to implement the rules and counters of Advanced Squad Leader. **ASLInputTool** is the WPF editor for creating boards, counters, and scenarios.

## Projects

| Project | Description |
|---------|-------------|
| HexLib | Hex-grid geometry — coordinates, board layout, adjacency, pathfinding |
| ASL | ASL domain — units, components, rules pipeline, persistence |
| ASLInputTool | WPF editor — ViewModels, converters, infrastructure |
